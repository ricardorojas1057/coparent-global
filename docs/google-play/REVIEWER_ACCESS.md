# Acceso para revision de Google Play

## Recorrido sugerido

1. Iniciar sesion con la cuenta de revision.
2. Abrir Inicio y verificar el resumen familiar.
3. Abrir Calendario y crear un evento.
4. Abrir Mensajes y enviar un mensaje de prueba.
5. Abrir Gastos y registrar un gasto compartido.
6. Abrir Perfil para revisar idioma, privacidad y eliminacion de cuenta.

## Antes de enviar

- Crear una familia de demostracion sin datos reales.
- Crear dos cuentas exclusivas para revision y verificar sus credenciales.
- No usar datos de hijos reales, conversaciones reales ni informacion financiera real.
- Mantener las cuentas activas durante todo el proceso de revision.
- Entregar a Play Console las credenciales y cualquier instruccion especial.
- No escribir credenciales reales dentro de este archivo ni subirlas a GitHub.
- Verificar que las cuentas de revision no tengan autenticacion multifactor que impida el acceso de Google.

## Nota sugerida para Play Console

La aplicacion requiere una cuenta para proteger los espacios familiares. Las
credenciales de revision proporcionadas acceden a una familia ficticia con
datos de demostracion. Desde Inicio se puede navegar a Calendario, Mensajes,
Gastos y Perfil. No se requiere realizar ninguna compra para revisar las
funciones principales.
